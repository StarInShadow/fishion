# 裂变 \~fission\~

一个支持关卡编辑、连锁消除、激光特效与音效的数字棋盘游戏。

## 玩法简介
- 点击怪物使其血量减少，血量归零时触发激光连锁。
- 十字怪（C）发射十字激光，X怪（X）发射对角激光。
- 支持关卡模式、随机挑战、每日挑战。
- 步数限制、撤销、重玩、音效开关等功能。
- 关卡数据采用序列化格式，支持自定义棋盘大小。

## 文件说明
- `index.html`：主页面结构
- `style.css`：样式表，支持动态棋盘
- `fission.js`：主逻辑，关卡解析、渲染、音效、动画
- `README.md`：项目说明
- `LICENSE`：MIT License

## 运行方式
直接用浏览器打开 `index.html` 即可游玩。
